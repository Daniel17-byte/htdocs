In acest fisier avem fiserele cu extensia php care reprezinta niste pagini web sau functii care fac anumite procese.
Pentru a rula web site ul avem nevoie de un server xampp care sa se conecteze la apache si la mysql workbench sau phpMyAdmin.
Fisierele care vor fi accesibile pe site le gasiti in xampp in htdocs.
Dupa ce acestea au fost facute putem accesa site-ul pe un browser accesand link ul localhost/main.php.
Pe acest site puteti sa va faceti un cont si puteti comanda produse cat si sa va vizualizati comenzile.
